# ⛵ sailr

*an open-waters sailing adventure with a real wind tunnel under the hood*

**Play it:** grab a build from [Releases](https://github.com/antipolaire/fluidr-game/releases) —
Windows, macOS (Apple Silicon & Intel), Linux.

**See it:** the landing page lives at
[antipolaire.github.io/fluidr-game](https://antipolaire.github.io/fluidr-game/).

Unzip, keep the `assets` folder next to the binary, run it. macOS builds are
unsigned — right-click → *Open* the first time. Saves and your logbook live
in `~/.sailr`.

The boat you steer isn't animated to feel right: its performance is solved
at launch by a velocity-prediction program from the wind-tunnel research
code the game grew out of. That source lives in a private research
repository for now; this repo hosts the game's releases and its page.
Wishes, wreck reports and feedback → [Issues](https://github.com/antipolaire/fluidr-game/issues).

*the chart is earned.*
